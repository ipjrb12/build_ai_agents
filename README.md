# Building AI Agents with LangChain
 
![Python](https://img.shields.io/badge/Python-3.9+-blue?style=flat-square&logo=python)
![LangChain](https://img.shields.io/badge/LangChain-latest-blueviolet?style=flat-square)
![License](https://img.shields.io/badge/License-MIT-green?style=flat-square)
 
Learning project exploring LangChain's agent framework — tool use, ReAct-style reasoning loops, and multi-step task execution with LLMs.
 
## What's here
 
Notebooks and scripts covering agent construction with LangChain: defining tools, building agent executors, handling intermediate steps, and connecting agents to external APIs and search.
 
## Running
 
```bash
pip install langchain langchain-openai openai python-dotenv
```
 
Set your API key in `.env`:
 
```
OPENAI_API_KEY=your_key_here
```
 
Then run the notebooks or scripts directly.
